Visualization

The SkyWalking native UI provides a default solution for visualization. It provides observability related graphs on overview, service, service instance, endpoint, trace, and alarm, such as topology maps, dependency graphs, heatmaps, etc.

We know that many of our users have integrated SkyWalking into their own products. If you would like to do that too, please refer to the SkyWalking query protocol.
